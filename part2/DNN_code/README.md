## 初期化手順
```
pip install -r requirements.txt
jupyter notebook
```

上記でうまくいかない場合は下記コマンドで各個インストール
```
pip install python-mnist
pip install Pillow
pip install matplotlib
pip install numpy
pip install pandas
pip install jupyter
pip install tensorflow
pip install sklearn

```
